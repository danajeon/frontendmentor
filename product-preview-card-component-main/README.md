### What I learned

This project strengthened my understanding of how CSS layout systems behave, especially when working with Flexbox and Grid together. 

* **Width vs. max-width and responsive behavior:** I gained a better understanding of when to use ``width`` vs. ``max-width``. Using ``width: 100%`` can remove natural spacing, while ``max-width`` allows layouts to stay responsive without stretching unnecessarily. 

* **Spacing with Grid and the importance of ``gap``:** I discovered that container-based spacing is more effective than manually adjusting individual element padding or margins. Using ``gap`` in both grid and flex containers made spacing consistent and easier to manage across different screen sizes.

* **Flexbox alignment:** I learned that even small elements like icons in buttons can behave unexpectedly if flex alignment isn't set explicitly. 

### Continued development

* **Deeper understanding of layout systems:** I want to continue improving my ability to choose between Flexbox and Grid and understand how they interact. THis project showed me that many layout issues come from default behaviors rather than obvious mistakes.

* **Mastering spacing and layout flow:** I'd like to get more comfortable using tools like ``gap``, ``margin``, and ``padding`` appropriately, and relying more on container-based spacing instead of adjusting individual elements.
